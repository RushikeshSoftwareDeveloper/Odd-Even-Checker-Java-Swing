# 🔢 Odd Even Checker (Java Swing)

## 📌 Project Description
A simple GUI-based Odd Even Checker developed using Java Swing.
The user enters a number, and the system determines whether the number is odd or even.

## 🚀 Features
- User input using text field
- Odd / Even result display
- Input validation
- Simple and interactive GUI

## 🛠 Concepts Used
✔ Java Swing (JFrame, JLabel, JTextField, JButton)  
✔ Event Handling (ActionListener)  
✔ Input Validation  
✔ Object-Oriented Programming (OOP)

## ▶ How to Run
1. Clone the repository
2. Open the project in Eclipse / IntelliJ
3. Run the main class
